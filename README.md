# Open Desktop

> A desktop environment project with python!

**A open source desktop**


## Development Status

**Work in Progress**

The desktop environment is under heavy development. Expect breaking changes, incomplete features, and experimental code.

## Supported Platforms
* Linux

## Installation
- First install the setup.sh
- run it in the folder you wanna install our program
- open the Open-desktop folder that was made by the program
- Run desktop.py
